corepost-mobile-android это panic-клиент на Jetpack Compose и Material 3.
Приложение хранит server URL, emergencyId и panicSecret локально и позволяет их изменить или удалить.
Клиент поддерживает check, lock, confirm lock и unlock для отдельного инстанса сервера.
Release APK публикуется отдельно, а демонстрационные изображения и видео остаются в репозитории как артефакты.
Сетевой адрес не зашит в код и задаётся пользователем.
