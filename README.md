### corepost-mobile-android 
panic-клиент на Jetpack Compose и Material 3.
Приложение хранит server URL, emergencyId и panicSecret локально и позволяет их изменить или удалить.
Клиент поддерживает check, lock, confirm lock и unlock для отдельного инстанса сервера.
Release APK публикуется в релизах, демка лежит в docs
